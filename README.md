# eogRichExif
A eog (Eye of GNOME Image Viewer) plugin which shows many Exif info in side pane.

To install, put these files (.glade, .py, .plugin) in

  $XDG_DATA_HOME/eog/plugins/

Usually default value for $XDG_DATA_HOME is $HOME/.local/share

Need to install libexiv2, python3-dev, py3exiv2(http://python3-exiv2.readthedocs.org/en/latest/index.html).

Then enable it in eog Preferences, Plugins.
